# Advanced Data Management Project

This project focuses on building a relational data pipeline to analyze customer rental behavior for a DVD rental business. The goal is to generate actionable insights for targeted email marketing campaigns.

## Features
- Detailed and summary tables for customer rental analysis.
- Automated data refresh using triggers and stored procedures.
- SQL transformations including field concatenation, aggregation, and indexing.
- Efficiently identifies top customers and supports marketing decision-making.

## Technologies Used
- PostgreSQL
- SQL (queries, triggers, stored procedures)
- Data analysis for marketing insights
