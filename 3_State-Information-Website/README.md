# State Information Website

This project is a multi-page website providing information about Ohio, including city-specific pages and interactive features.

## Features
- Multiple pages for different cities.
- Interactive and validated web forms.
- Responsive layouts and accessible design principles.
- Semantic HTML for improved usability.

## Technologies Used
- HTML5 / CSS3
- JavaScript
- Responsive and accessible web design


## Notes
- Developed as part of software engineering coursework to demonstrate front-end web development and interactive web design skills.
